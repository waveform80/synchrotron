Model Synchrotron

This project is about building a model of a particle acclerator, specifically the Diamond Radiation Light Source. The model will hopefully use a ball-bearing to represent the photons in the ring, with holding solenoids used to draw it around. LED strips will be used to represent the beam-lines, and both the accelerator and storage rings will be represented.

The following software was used in this project:

  • Blender - for 3D modelling
  • OpenSCAD - for 3D to 2D projections
  • Inkscape - for post-processing of the 2D projection
  • Ubuntu Linux - everything was run under Ubuntu Linux (specifically 12.04 LTS)

The following files were produced as part of this project:

  • synchrotron.blend - a 3D model of the project built in Blender
  • synchrotron.stl - an export of the model in STL format for further processing (to generate this yourself, open synchrotron.blend in Blender, select the main model, select File / Export / STL)
  • synchrotron.scad - an OpenSCAD script used to take 2D slices from the model for ShopBot
  • synchrotron_top.dxf - a slice through the top of the model in Autodesk DXF format
  • synchrotron_bottom.dxf - a slice through the bottom of the model in Autodesk DXF format
