FabLab Synchrotron Project

This project is about building a model of a particle acclerator, specifically the Diamond Radiation Light Source. The model will hopefully use a ball-bearing to represent the photons in the ring, with holding solenoids used to draw it around. LED strips will be used to represent the beam-lines, and both the accelerator and storage rings will be represented.

The purpose of the model is demonstrate to the public the operation of synchrotron radiation light-sources as they relate to some of the work carried out by the Manchester Organic Geochemistry Group.

Current status: planning

The following software was used in this project:

  • Blender - for 3D modelling
  • OpenSCAD - for 3D to 2D projections
  • Inkscape - for post-processing of the 2D projection
  • Ubuntu Linux - everything was run under Ubuntu Linux (specifically 12.04 LTS)

The following files were produced as part of this project:

  • synchrotron.blend - a 3D model of the project built in Blender
  • synchrotron.stl - an export of the model in STL format for further processing. The method for generating this is:
    • open synchrotron.blend in Blender
    • select the main model (right click on the object)
    • select File / Export / STL
  • synchrotron.scad - an OpenSCAD script used to take 2D slices from the model for ShopBot
  • synchrotron_top.dxf - a slice through the top of the model in Autodesk DXF format. The method for generating this is:
    • open synchrotron.scad in OpenSCAD
    • ensure the second projection line is commented
    • select Design / Compile and Render (F6)
    • select Design / Export STL
  • synchrotron_bottom.dxf - a slice through the bottom of the model in Autodesk DXF format. The method for generating this is the same as synchrotron_top above, but ensure the first projection line is commented before compiling and rendering as STL.
  • synchrotron_top.svg - SVG conversion of the DXF file. The method for generating this is:
    • import synchrotron_top.dxf in Inkscape
    • ungroup the imported object (Ctrl+Shift+G)
    • select Path / Combine while all objects are still selected
    • switch to the node tool (F2), and press Ctrl+A to select all nodes
    • click Join Selected Nodes on the toolbar; at this point instead of having lots of separate lines you'll now have one coherent path with several subpaths but the directions of the sub-paths will be incorrect for filling
    • starting at the outer-most subpath, select a single node on the subpath within it and select Path / Reverse; continue doing this for each subpath working your way inwards
    • finally, switch back to object selection (F1), open the Fill and Stroke toolbox and apply a fill
  • synchrotron_bottom.svg - an SVG conversion of the synchrotron_bottom.dxf file. This is produced in the same way as the top version.

All files in this project are also available from GitHub.

We are extremely grateful to the helpful community at FabLab Manchester for their help so far. Hopefully with a bit more, we can build something pretty impressive!
